# Code Quest

Code Quest is a daily programming challenge platform built to make learning programming a habit.

Every day, users receive a new AI-generated multiple-choice programming question covering topics such as JavaScript, TypeScript, Python, C#, SQL, algorithms, and more. The long-term vision is to expand the platform with coding challenges, interview simulations, leaderboards, streaks, and personalized learning paths.

## Features
- 🤖 AI-generated programming questions
- 🎯 Four-answer multiple-choice format
- 📅 New challenge every day
- 📈 Planned streak and progress tracking
- 💻 Future coding challenges and interview mode

Built as a full-stack project using modern web technologies.
